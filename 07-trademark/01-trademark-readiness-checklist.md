# OCD Trademark Readiness Checklist

This checklist organizes facts for trademark counsel. It is not a clearance opinion or filing authorization.

## Search

- [ ] Exact OCD search
- [ ] Exact Obsessive Cannabis Disorder search
- [ ] Similar spelling and sound search
- [ ] Related goods/services search
- [ ] State databases and common-law use reviewed
- [ ] Domain and social use reviewed
- [ ] Design/logo search reviewed

## Ownership

- [ ] Correct applicant/owner identified
- [ ] Founder assignment complete
- [ ] Designer rights complete
- [ ] Goodwill included
- [ ] No conflicting prior filing
- [ ] Any ownership correction or assignment record prepared

## Goods and Services

- [ ] Lawful goods/services identified
- [ ] Cannabis-related federal-law issue reviewed by trademark counsel
- [ ] State-level strategy reviewed
- [ ] Non-cannabis goods/services not invented merely to obtain registration
- [ ] Specimens are accurate and not fabricated

## Filing

- [ ] Filing basis approved
- [ ] Mark format approved
- [ ] Owner address approved
- [ ] Public-contact implications reviewed
- [ ] Attorney representation determined
- [ ] Search and filing report retained
