# Upload This Repository to GitHub

## Best Method — GitHub Desktop

1. Create or open the private GitHub repository.
2. In GitHub Desktop, choose **File → Clone repository**.
3. Clone the OCD repository to your computer.
4. Extract this ZIP.
5. Open the extracted `flora-client-ocd-complete` folder.
6. Copy everything inside it into the cloned GitHub repository folder.
7. Return to GitHub Desktop.
8. Enter commit message: `Build complete OCD brand foundation repository`.
9. Click **Commit to main**.
10. Click **Push origin**.

This preserves all folders, including `.github/workflows`.

## Browser Method

The GitHub browser uploader may flatten folders depending on how files are dragged. Git also does not track empty folders.

Use GitHub Desktop for this package.
