# Exhibit B — Logo and Icon Specimen Schedule

## Specimen Requirements

For each logo or icon, record:

- exact file name;
- SHA-256 hash;
- format;
- dimensions;
- creator/designer;
- creation date;
- first commercial use;
- claimed owner;
- assignment/license record;
- approved uses; and
- prohibited or disputed elements.
